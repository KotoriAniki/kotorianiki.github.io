Vue.jsの使い方

ディレクティブ

v-*といった具合に属性を指定する機能。カスタムディレクティブの定義も可能。

二重波括弧でテキストを表示

表示

{{ message }}

HTML

<div id="app-1">{{ message }}</div>

script

var app1 = new Vue({
  el: "#app-1",
  data: {message: "Hello World!"}
})

説明

Vue.jsが適用された範囲内では、{{message}}などと表記することで、
JavaScript側で保持する変数の値に置き換えられる。

v-textでテキストを表示する

表示

HTML

<div id="app-101">
  <div v-text="message"></div>
</div>

script

var app101 = new Vue({
  el: "#app-101",
  data: {message: "Hello!"}
})

説明

{{}}と同じ役割。

v-htmlでhtmlとして表示する

表示

HTML

<div id="app-102">
  <div v-html="message"></div>
</div>

script

var app102 = new Vue({
  el: "#app-102",
  data: {message: "<b>Hello!</b>"}
})

説明

通常のテキストは<, >, &などは、<, >, &に置換されて出力されるが、
v-htmlでは置き換えをせずに出力する。
ただしこれはセキュリティの面からも、クロスサイトスクリプティングに注意して
使う必要がある。

v-showで条件に応じて表示を制御する

表示

Hello!

HTML

<div id="app-103">
  <input type="checkbox" @click="change" checked>
  <span v-show="seen">Hello!</span>
</div>

script

var app103 = new Vue({
  el: "#app-103",
  data: {seen: true},
  methods: {
    change: function(e) {
      this.seen = e.target.checked
    }
  }
})

説明

v-else-ifで条件に応じて表示を制御する

表示

Good morning. Hello! Bye!

HTML

<div id="app-106">
  <input type="radio" name="app106-type" onclick="app106.type='A'" checked>
  <input type="radio" name="app106-type" onclick="app106.type='B'">
  <input type="radio" name="app106-type" onclick="app106.type='C'">
  <span v-if="type=='A'">Good morning.</span>
  <span v-else-if="type=='B'">Hello!</span>
  <span v-else>Bye!</span>
</div>

script

var app106 = new Vue({
  el: "#app-106",
  data: {type: 'A'}
})

説明

v-forでループする

表示

  • {{color}}

HTML

<div id="app-107">
  <ul>
    <li v-for="color in colorList">{{color}}</li>
  </ul>
</div>

script

var app107 = new Vue({
  el: "#app-107",
  data: {colorList: ["Red", "Green", "Blue"]}
})

説明

v-on(@)でイベントを処理する

表示

HTML

<div id="app-108">
  <button v-on:click="hello">Hello</button>
  <button @click="hello">Hello</button>
</div>

script

var app108 = new Vue({
  el: "#app-108",
  methods: {
    hello: function() {
      alert("Hello!");
    }
  }
})

説明

表示

script

v-cloakで二重波括弧の表示を防ぐ

表示

Message: {{message}}
Message: {{message}}

HTML

<div id="app-122">
  <button onclick="location.reload()">再表示</button>
  <div>Message: <span v-cloak>{{message}}</spna></div>
  <div>Message: <span>{{message}}</spna></div>
</div>

script

window.setTimeout(function() {
  var app122 = new Vue({
    el: "#app-122",
    data: {message: "Hello!"}
  })
}, 1000)

コンポーネントで独自要素を定義する(Vue.component())

表示

HTML

  <my-element-140></my-element-140>
</div>

script

  template: '<strong>Hello</strong>'
})

  el: '#app-140',
})
